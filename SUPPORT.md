# Support

## How to get help

### I have a question
If you have questions about how to use Devonz, please utilize the following resources:
1.  Read the [Documentation](./README.md).
2.  Check [ARCHITECTURE.md](./ARCHITECTURE.md) for technical details.
3.  Search existing [GitHub Issues](https://github.com/kherrera6219/Devonz/issues).

### I found a bug
If you believe you have found a bug, please:
1.  Search existing issues to see if it has already been reported.
2.  If not, open a new issue using the **Bug Report** template.
3.  Provide as much detail as possible, including steps to reproduce.

### I have a feature request
We welcome feature suggestions! Please use the **Feature Request** template to submit your idea.

## Commercial Support
For commercial support inquiries, please contact the maintainers directly.
